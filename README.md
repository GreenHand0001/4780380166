# 项目简介

## 核心功能
- 基于YOLOv8的铸件表面缺陷检测，支持5类缺陷识别：Casting_burr、crack、scratch、pit、Polished_casting
- 提供完整源码、2500张标注数据集、训练好的模型及训练结果
- 支持图片、视频及摄像头三种输入方式进行检测
- 支持自定义置信度和IOU阈值
- 提供UI界面，可实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 可选择不同检测模型

## 适用场景
- 铸件制造行业表面质量检测
- 工业生产线自动化质检
- 材料缺陷研究与分类
- 制造业质量控制流程

## 技术特点
- 采用Python 3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等环境
- 使用PyCharm+Anaconda或VSCode+Anaconda作为开发环境
- 提供完整UI界面，操作简便直观
- 已配置成功环境，可直接使用
